# JioHotstar

A Node.js video streaming application built with Express.js and EJS templating.

## Features

- Video upload and streaming
- Static file serving
- EJS templating engine
- Express.js backend

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```

## Usage

The application runs on `http://localhost:8000` by default.

## Project Structure

- `server.js` - Main application file
- `config/` - Database configuration
- `routes/` - Application routes
- `views/` - EJS templates
- `public/` - Static assets
- `uploads/` - Uploaded files

## Technologies Used

- Node.js
- Express.js
- EJS
- MongoDB (via db.config)
